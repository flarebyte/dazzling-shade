> Artistic module to manipulate colours


## Install

```sh
$ npm install --save dazzling-shade
```


## Usage

```js
var dazzling-shade = require('dazzling-shade');

dazzling-shade('Rainbow');
```

```sh
$ npm install --global dazzling-shade
$ dazzling-shade --help
```

```sh
# creates a browser.js
$ npm run browser
```
